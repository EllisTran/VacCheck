# VacCheck Web

To start up VacCheck Web 

### `cd ./VacCheckWeb`
### `npm install`

To run development enviornment run

### `npm start`

# VacCheck Mobile

To start up with VacCheck Mobile

### `cd ./VacCheckMobile`
### `flutter pub get`

### `cd ./ios`
### `pod install`
### `cd ..`
### `flutter run`
